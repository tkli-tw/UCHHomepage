1. 新增一個 repository
-----------------------------------------------

先移進一個目錄，然後打，
>> git init


2. 查看目前 git 的狀態
-----------------------------------------------
>> git status


3. 新增 git 要追蹤的目標
-----------------------------------------------
>> git add <filename>

可用
>> git add *
新增該目錄內所有檔案

注意一下，git 並不會自動 將追蹤目標的更新 納入，
當 git add 某檔案後，若該檔案又修改了，請再做一次 git add ，這樣才能將該檔案的最新的更新納入。
